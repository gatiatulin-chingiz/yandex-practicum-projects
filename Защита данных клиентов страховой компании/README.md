# Защита данных клиентов страховой компании

## Данные

В наличии были следующие данные:
- Пол клиента
- Возраст клиента
- Заработная плата
- Количество членов семьи
- Страховые выплаты

## Задача
Разработка модели анонимизации персональных данных.

## Используемые библиотеки
*pandas*, *numpy*, *sklearn*, *matplotlib*
